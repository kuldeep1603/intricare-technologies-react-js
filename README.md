⚡ Installation & Setup

Clone the repository:

git clone https://github.com/kuldeep1603/intricare-technologies-react-js.git
cd intricare-technologies-react-js

Run Figma Design
cd figma-design
npm install
npm run dev

Run Dashboard Product Management
cd Product\ Management
npm install
npm run dev
